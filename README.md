# Ahmednagar Shahar Sahakari Bank — Website

Modern website for [Ahmednagar Shahar Sahakari Bank Ltd.](https://new.shaharbank.com),
a cooperative bank in Ahmednagar and Pune, established 1970.

## Stack
- React 19 + Vite 8
- Tailwind CSS v4
- Framer Motion (animations)
- React Router v7
- react-helmet-async (SEO)

## Setup
npm install
npm run dev

## Deploy (cPanel)
npm run build
# Upload contents of /dist/ to public_html/
# Ensure .htaccess is present in public_html/
